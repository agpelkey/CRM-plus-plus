# CRM-plus-plus
API to handle client managemenet for my girlfriend's real estate business.

## Features
- [X] REST API written with httprouter.
- [X] PostgreSql database provider.
- [X] Dockerized.

## How to run locally
Clone the repo locally and run:
```shell
$ make run
```
